# 貢獻指南

感謝您有興趣為此專案做出貢獻！

## 如何貢獻

### 回報問題

1. 確認問題尚未被回報
2. 使用 Issue 模板描述問題
3. 提供重現步驟和環境資訊

### 提交 Pull Request

1. Fork 此專案
2. 建立功能分支：`git checkout -b feature/your-feature`
3. 提交變更：`git commit -m 'Add some feature'`
4. 推送分支：`git push origin feature/your-feature`
5. 開啟 Pull Request

### 程式碼規範

- 遵循 PEP 8 風格指南
- 為新功能撰寫文件
- 確保所有測試通過

### 資料集貢獻

如果您想擴充訓練資料：

1. 確保對話符合現有格式（JSONL）
2. 對話應為繁體中文（台灣）
3. 涵蓋多樣化的點餐情境
4. 避免包含個人資訊

## 行為準則

- 尊重所有貢獻者
- 保持友善和專業的討論
- 接受建設性的批評

## 授權

貢獻的內容將採用與專案相同的 Apache 2.0 授權。
